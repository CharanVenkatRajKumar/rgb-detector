RGB-Value Detector

Project Brief

  • “RGB Value Detector” is an application that can detect the color from the image uploaded and give out the name in both text and voice command form.
  • The image uploaded by user is processed and identified with precise name from database and given out in the form of voice command
  • The color codes are also displayed along with name to help graphic designers to identify the color codes to use in their work
